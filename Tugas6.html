PageRank#

Pengertian PageRank#

PageRank adalah algoritma yang dikembangkan oleh Larry Page dan Sergey Brin, pendiri Google, sebagai bagian dari proyek penelitian mereka di Stanford University pada akhir 1990-an. Algoritma ini digunakan untuk mengukur pentingnya halaman web berdasarkan jumlah dan kualitas tautan yang menuju ke halaman tersebut. Secara sederhana, PageRank menilai sebuah halaman web sebagai lebih penting jika banyak halaman web lain yang penting menautkan ke halaman tersebut.

Prinsip Dasar PageRank#

  1. Tautan sebagai Suara: Setiap tautan dari satu halaman web ke halaman web lain dianggap sebagai “suara” untuk halaman yang ditautkan.

  2. Bobot Tautan: Tidak semua tautan memiliki bobot yang sama. Tautan dari halaman yang memiliki PageRank tinggi memberikan bobot lebih besar daripada tautan dari halaman dengan PageRank rendah.

  3. Distribusi PageRank: Total PageRank dari sebuah halaman dibagi secara merata di antara semua tautan keluar yang dimiliki halaman tersebut. Halaman yang menerima tautan dari banyak halaman dengan PageRank tinggi akan memiliki PageRank yang lebih tinggi.

  4. Iterasi Algoritma: PageRank dihitung melalui iterasi. Algoritma mulai dengan memberikan PageRank awal yang sama untuk semua halaman, kemudian menghitung ulang nilai PageRank berdasarkan tautan yang diterima, dan terus mengulangi proses ini hingga konvergensi (saat perubahan nilai PageRank menjadi sangat kecil atau stabil).

Formula Sederhana PageRank#

\[ [ PR(A) = (1 - d) + d \left( \sum_{i=1}^{n} \frac{PR(T_i)}{C(T_i)} \right) ] \]

Di mana: $\(- ( PR(A) )\)\( adalah PageRank dari halaman A. \)\(- ( d )\)\( adalah faktor damping (biasanya sekitar 0,85), yang memperhitungkan kemungkinan bahwa pengguna internet akan melompat ke halaman lain secara acak daripada mengikuti tautan. \)\(- ( T_i )\)\( adalah halaman yang menautkan ke halaman A. \)\(- ( C(T_i) )\)\( adalah jumlah tautan keluar dari halaman \)\(( T_i )\)\(. \)\(- ( \sum )\)\( menunjukkan penjumlahan atas semua halaman \)\(( T_i )\)$ yang menautkan ke halaman A.

Algoritma PageRank sangat berpengaruh dalam keberhasilan awal mesin pencari Google, karena memungkinkan untuk menyusun peringkat halaman web berdasarkan relevansi dan otoritas yang dihitung secara matematis. Meskipun algoritma pencarian Google saat ini jauh lebih kompleks dan menggunakan banyak faktor lain selain PageRank, konsep PageRank masih menjadi bagian dari dasar penilaian relevansi halaman web di internet.

Agar lebih paham: Node 1 mempunyai 3 sisi keluar, sehingga akan meneruskan \(\frac{1}{3}\)kepentingannya ke masing-masing 3 node lainnya. Node 3 hanya mempunyai satu sisi keluar, jadi node tersebut akan meneruskan seluruh kepentingannya ke node 1. Secara umum, jika sebuah node mempunyai k sisi keluar, maka node tersebut akan meneruskan \(\frac{1}{k}\) kepentingannya ke setiap node yang terhubung dengannya. Mari kita memvisualisasikan prosesnya dengan lebih baik dengan memberikan bobot pada setiap sisi.

\[\begin{split} \begin{pmatrix} 0 & 0 & 1 & \frac{1}{2}\\ \frac{1}{3} & 0 & 0 & 0\\ \frac{1}{3} & \frac{1}{2} & 0 & \frac{1}{2}\\ \frac{1}{3} & \frac{1}{2} & 0 & 0 \end{pmatrix} \end{split}\]
## implementasi dalam code

import numpy as np

def peringkat_halaman(matriks_adjacency, d=0.85, toleransi=1e-6, maks_iterasi=100):
    """
    Menghitung PageRank menggunakan metode iterasi kuasa.

    Parameter:
        matriks_adjacency (numpy.ndarray): Matriks adjacency yang mewakili graf web.
        d (float): Faktor redaman (default adalah 0.85).
        toleransi (float): Toleransi untuk menentukan konvergensi (default adalah 1e-6).
        maks_iterasi (int): Jumlah maksimum iterasi (default adalah 100).

    Returns:
        numpy.ndarray: Peringkat PageRank untuk setiap node.
    """
    n = matriks_adjacency.shape[0]
    # Inisialisasi skor PageRank
    p = np.ones(n) / n
    for _ in range(maks_iterasi):
        p_sebelumnya = p.copy()
        # Menghitung skor PageRank baru
        p = (1 - d) / n + d * matriks_adjacency.T.dot(p)
        # Memeriksa konvergensi
        if np.linalg.norm(p - p_sebelumnya, 1) < toleransi:
            break
    return p

# Contoh penggunaan dengan matriks yang diberikan
if __name__ == "__main__":
    # Graf web contoh (matriks adjacency)
    matriks_adjacency = np.array([[0, 0, 1, 1/2],
                                  [1/3, 0, 0, 0],
                                  [1/3, 1/2, 0, 1/2],
                                  [1/3, 1/2, 0, 0]])
    # Menghitung skor PageRank
    peringkat = peringkat_halaman(matriks_adjacency)
    print("Skor PageRank:", peringkat)
